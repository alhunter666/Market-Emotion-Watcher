# 项目核心业务愿景 (purpose.md)

## 建设目标
- 构建一个面向美股市场的核心情绪、波动率、资金轮动与异常流动性监控看板 MVP。

## 核心监控指标体系与多维对比规范

### 一、 衍生品与波动率体系（情绪预警）
1. **VIX 指数体系**：标普 500 期权隐含波动率。展示当日绝对值、较昨日变动（Today - Yesterday）、以及 5 日移动均线（5-day MA），用以捕捉短期市场情绪的恐慌突变（Spikes）。
2. **美债波动率（^VXTLT）**：长债期权隐含波动率。监控当日值、较昨日差值。若 VIX 平静但 VXTLT 飙升，判定为跨市场利率定价撕裂的潜在危机信号。
3. **隐含相关性指数（^COR1M）**：监控当日绝对值。当数值逼近 1.0 时（高资产相关性），触发“系统性流动性危机/无差别抛售”高能报警。
4. **期权交割日（OpEx）预警**：通过 Python 自动计算并定位当前月份的第三个周五，并在看板显著位置进行文字提示，警告做市商 Gamma/Charm 效应可能引发的日内及尾盘极端波动风险。

### 二、 市场结构与资金全景（健康度验证）
5. **【今日美股11板块全景雷达】**：一键式直接抓取 11 个核心行业 ETF（XLK, XLF, XLY, XLE, XLV, XLP, XLU, XLB, XLI, XLRE, XLC）的当天百分比涨跌幅。前端展示彻底废弃不稳定的 HTML 表格，强制降级使用 Streamlit 原生的 `st.metric` 组件进行网格排布展示，保证绝对稳定性。
6. **【GEX 冰山雷达图（GEX 风险管理）】**：读取主力做市商在当前指数点位的绝对 Gamma 暴露风险水位（gex_last），进行 GEX 风险管理。在技术架构上正式废弃 `st.bar_chart` 图表和 HTML/CSS 进度条，全面改为使用原生的 `st.progress` 进度条进行 0.0 到 1.0 的归一化水位渲染。
7. **McClellan Oscillator（麦克莱兰摆动指标）**：基于纽交所（NYSE）上涨与下跌家数差值计算其指数移动平均（EMA），展示当前数值，并计算其与 ±50 和 ±100 短期动能衰竭红线的实时绝对偏离距离。

### 三、 宏观与暗流体系（底层资金支撑）
8. **【宏观信用与流动性矩阵（美联储三因子 3 年滚动历史分位数）】**：基于美联储总资产、TGA账户余额和隔夜逆回购规模这“美联储三因子”计算美元净流动性【WALCL / 1000 - WTREGEN - RRPONTSYD】，并滚动计算过去 3 年历史滚动分位数（10%极枯竭水位 / 90%极充沛水位）。当最新值跌入 10% 枯竭分位数以下时，界面触发 #F7ECE9 底色、红字警告的交易杠杆控制提示，以防御系统性流动性踩踏。
9. **高收益债利差（OAS）**：调用 FRED 代码【BAMLH0A0HYM2】（ICE BofA 美银美林高收益债期权调整利差）。展示今日利差与 20 日均线，监控股市底层信用环境的恶化及风险偏好收缩。
10. **暗池交易指数（DIX）**：抓取 SqueezeMetrics 免费 CSV（暗池买入比例）。当美股大盘暴跌时，若 DIX 指数逆势突发性飙升，判定为机构在暗池中悄悄抄底的铁证。
