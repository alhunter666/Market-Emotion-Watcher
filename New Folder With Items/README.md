# 项目沙盒交付说明书

本说明书用于指引量化交易看板 MVP 的运行部署，以及明确人机协作过程中的审计点。

## 环境依赖说明
本系统完全依赖本地 Python 原生数据清洗代码及以下第三方库：
- `yfinance`：用于抓取实时和历史美股、板块 ETF 及指数行情数据。
- `pandas`：用于高性能时间序列对齐、合并和统计指标计算。
- `streamlit`：用于构建前端可视化交互看板。
- `python-dotenv`：用于加载本地隔离的环境变量配置。

## 人机协作审计卡点指引
- **协作规范**：根据 Session 3 协作规范，开发流程中的每一步关键变更都必须输出 Diff 审计与操作总结（Walkthrough）。
- **终审确认**：所有重大架构调整、代码发布、策略参数调整的质量把关权牢牢交还给人类督导（Gatekeeper）。未获得 `Proceed` 授权前，智能体不得擅自修改核心逻辑。
